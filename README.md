## Features

- Fullscreen mode.
- Cross platform.
- Cloud backed-up.
- Pick image from gallery.
- Upload image to firebase.
- Retrieve image from firebase.
- Download Image to device.
- Send mesaages immediately 
- Receive notifications on mesaages
